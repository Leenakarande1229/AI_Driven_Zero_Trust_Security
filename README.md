# AI Driven Zero Trust Security

This project is a Flutter-based smart utility application that demonstrates AI-assisted security concepts using rule-based anomaly detection.

## Features
- Detects suspicious traffic patterns
- Implements anomaly-based logic
- User-friendly Flutter UI
- Prototype for Zero Trust Security model

## Tech Stack
- Flutter
- Dart
- Basic AI Logic (Rule-based)

## Description
This project simulates how AI can assist in identifying unusual behavior in a system and improve security decision-making.
